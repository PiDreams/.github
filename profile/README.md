Hey! We're PiDreams.

We're a small group of developers who enjoy building projects for Raspberry Pi devices. Most of our work focuses on creating software that's simple, lightweight, and easy to run without needing powerful hardware or complicated setup processes.

A lot of software today is built with high-end computers in mind. Raspberry Pis are different. They're small, affordable, and surprisingly capable, so we believe the software made for them should be efficient and straightforward too.

## What You'll Find Here

Our projects usually fall into a few categories:

* Raspberry Pi utilities
* Home and hobby automation tools
* Touchscreen applications
* LED and GPIO projects
* Monitoring and dashboard software
* Small experiments and fun ideas

Some projects are practical, some are just things we thought would be fun to build.

## Our Goal

We want to create software that:

* Starts quickly
* Uses minimal resources
* Is easy to install
* Doesn't require a huge list of dependencies
* Works well on real Raspberry Pi hardware

Whether you're using a Raspberry Pi Zero, a Pi 4, or something newer, we try to keep our projects accessible.

## Open Source

Everything we build is open source. Feel free to explore the code, suggest improvements, open issues, or contribute if you'd like.

Even simple feedback helps us improve future projects.

## Current Projects

### PiSleepTimer

A touchscreen sleep timer for Raspberry Pi that controls WS2812B and NeoPixel LED strips. It includes customizable timers, brightness controls, color transitions, and a simple fullscreen interface designed for touchscreens.

More projects are planned and will be added over time.
